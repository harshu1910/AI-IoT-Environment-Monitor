# 🌡️ AI Smart Environment Monitor

## Overview
This project is a local AI-powered system that analyzes temperature and humidity data and provides intelligent recommendations using a local LLM.

## Features
- Real-time simulated sensor data
- AI-generated suggestions (Ollama - phi3)
- Interactive Streamlit dashboard
- Works offline (no internet required)

## Tech Stack
- Python
- Streamlit
- Ollama (Local LLM)

## How to Run
1. Run: ollama run phi3
2. Run: streamlit run app.py
3. Open browser: http://localhost:8501

## Output
Displays temperature, humidity, and AI-based recommendations in a clean dashboard UI.
